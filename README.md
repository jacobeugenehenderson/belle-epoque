# Belle Epoque

Interactive 3D visualization of Downtown Belleville, IL.

## Development

```bash
npm install
npm run dev
```

## Deployment

This project deploys to its own GitHub Pages site, NOT to jacobhenderson.studio.

```bash
./deploy.sh
```

This builds the project and pushes to the `gh-pages` branch.

**Live site:** https://jacobeugenehenderson.github.io/belle-epoque/

**Source repo:** https://github.com/jacobeugenehenderson/belle-epoque
